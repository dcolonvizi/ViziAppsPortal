The View Reports page allows you to view graphs and reports of call session logs.

To start, select an available ViziApps app from the application drop down list on top.

Then select which type of graph or report you want to see.

  • The Call Volume Graphs show call volume graphs for defined time spans. To see a graph, set the defined the time span either from a list of pre-defined time spans or enter the start date-time and the end date-time. Then select the time span unit for the graph. Finally click on Show Graph to view a graph. After a graph is displayed you can further download the raw data in the graph in either XML or Excel by clicking the on corresponding button below the graph.

  • The Inbound Event Reports show Excel sheets detailing the call session events that occurred during a defined time span of inbound applications. To see a report, set the defined the time span either from a list of pre-defined time spans or enter the start date-time and the end date-time. Then click on Show Report to view the report.

  • The Outbound Event Reports show Excel sheets detailing the call session events that occurred for a defined outbound call-campaign run. To see a report in Excel, click on this icon . To  delete a campaign run click on this icon .
