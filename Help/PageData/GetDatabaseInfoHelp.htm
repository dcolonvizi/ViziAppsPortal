To interface with your back end database, you can visually create SQL statements for your app.

When you push a button or table row on your phone or iPad, your device will call the ViziApps Database Web Service located on your server with the URL that you provide. This Web Service will use a connection string  to your database on your server, that you also provide (more on this below)  to execute the SQL commands on your database .

You need to install the ViziApps Database Web Service , available for free, on your server (see below).

The app needs to know what type of database you are using, the schema of your database and a connection string to access your Database.

To provide this information for your app:

  • Select your Database Type
  • Generate a .SQL file, without data,  from the normal database manager you use for the database you want to access and Upload the .SQL file, so ViziApps gets the schema of your database. This will be used to automatically populate fields in creating SQL Commands.
  • Enter your database connection string that will allow the free web service, that we provide you, to access your database. If you have a firewall, provide the connection string from inside your firewall. For the syntax of the connection string click here. Here is an example connection string: DRIVER={MySQL ODBC 3.51 Driver};SERVER=localhost;UID=admin;PASSWORD=password; DATABASE=my_database;option=3;
  •  Click Save and then click Close

To download the free ViziApps Database Web Service for installation on your server click on DB Web Service. After you install this Web Service on your server, you need to provide the URL for your app on your device to directly access the Web Service  The download gives you the source code for a project of the ViziApps Database Web Service in C# ASP.NET that can be built using Visual Studio. Using this web service can be made secure using https access.
