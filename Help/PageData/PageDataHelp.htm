The Page Data page allows you to set and get data between your device screen on this page and your back end database. You can access data in 3 ways.

In the Google Docs SpreadSheet section, you can map the device fields to the columns in your Google Docs WorkSheets, which work just like Excel. Interface commands on this page let you select, insert, update and delete data to and from your phone. A graphical command tool lets you create the commands and choose the device fields and the worksheet fields for the commands. These fields are pre-populated after you enter the credentials to access your Google Docs account. 

In the Database section, you can map the device fields to your database tables and fields. To use this you will need to know what a SQL file is, how to generate it and know about SQL statements.

In the Web Services section, you can map the device fields to the parameters of your web service methods. To use this you will need to know what a web service is and how to access yours. Please read the web service sepcifications by clicking on the Web Service Specs  in the Web Service tab .
