Before creating the spreadsheet, consider the structure and sequence of getting and setting the data as you’ll refer to it in the app and try to create a similar structure in the spreadsheet. Also try to use identical or similar internal field names in the app and field names in the spreadsheet to simplify mapping the fields from app to spreadsheet.

  • Login to either your Google Docs or Googlel Apps account. Go to Google Docs page.

  •  Click the Create New button in the upper left and select Spreadsheet. A new browser window displays, with a blank spreadsheet.

  • Click the Save Now button in the upper right corner. The Rename Spreadsheet dialog box displays.

  •  Change the name to whatever you want.

  • Change the worksheet name at the bottom to whatever you want.

  •  In the first row of cells enter fields names. These field names should be lower case with no spaces.

  •  In other rows, enter your data. If you ever want to insert data into the Spreadsheet it must already contain some data to start.

  • Click the Share button in the upper right corner of the screen. The Sharing Settings dialog box displays.

  •  Click Change in the Permissions section. The Shared Settings dialog box displays.

  •  Select the Public On the WebAllow Anyone to Edit option, then click Save. The Shared Settings dialog box displays again.

  • Click Close. Select File > Save. Repeat the steps for other worksheets..
