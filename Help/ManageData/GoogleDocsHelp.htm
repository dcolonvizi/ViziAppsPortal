To interface your app with your Google Docs spreadsheet, you can create query commands for your app that will run when you tap a button or table row on your phone or iPad.

To create the interface from your phone or iPad to your database:

  • Create a Google Docs account or Google Apps account, if you don't already have one. Then create a spreadsheet with one or more worksheets. Click here for a step by step guide. Set your worksheet(s) with field names in the first row, your data and set the sheet name(s). Note that spaces and "_" will be ignored in the field names and worksheet names.
  • Click Connect to Your Google Docs. See the context help in the popup for more information.
  • Select the Event on the phone or iPad (some button or table field name) that will activate the query commands you are going to create.
  • Create query commands. Start by clicking on the +.
  •  Click Save and select other events and create their associated query commands.
