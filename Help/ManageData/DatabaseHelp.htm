To interface your app with your back end database, you can create SQL statements for your app that will run when you tap a button, switch or table row on your phone or iPad.

To create the interface from your phone or iPad to your database:

  • Click Get Database Info. See help in the popup for more information on how the information will be used.
  • Select the Event on the phone or iPad (some button or table field name) that will activate the SQL statements you are going to create.
  • Create SQL statements. Start by clicking on the +.
  •  Click Save and select other events and create their associated SQL statements.
  •  After you have finished all your SQL Commands, click DB Web Service  to download and install the ViziApps Web Service on your server and enter the URL for the installed Web Service on your server.
  • To download the free ViziApps Database Web Service for installation on your server and to provide the URL for your app to access the Web Service click on DB Web Service. The download gives you the source code for a project of the ViziApps Database Web Service in C# ASP.NET that can be built using Visual Studio. Using this web service can be made secure using https access.
