This is a reference to ViziApps javascript functions that can be used in custom header HTML to enhance the operation of your app. Your app runs within a jQuery and jQuery-Mobile javascript context. All of these functions need to be inside of the following javascript code:

<script type="text/javascript">
  $(function () {
  
     ... your code here ....
   
  });
</script>

The line $(function () is required for the JQuery context.

To run your code when a button or image button is tapped use this type code:

<script type="text/javascript">
$(function () {
   $('#button1').bind('tap',function(event){
     ... your code here ....
      event.stopPropagation();
   });
});
</script>

Here is the list of ViziApps javascript functions

getFieldValue('field_name')

Returns the string value for the ViziApps field

getFieldArrayValues('field_name')

Returns the array of string values for the ViziApps field

setFieldValue('field_name', field_value)

Sets the value for the ViziApps field

setFieldArrayValues('field_name', array_values)

Sets the array of string values for the ViziApps field

gotoPage('page_name')

Performs a page transition to the named page

getFieldType('field_name')

Returns the ViziApps field type of the field. The supported field types are:

  • button
  • image_button
  • label
  • text_field
  • text_area
  • image
  • audio
  • table
  • picker
  • slider
  • web_view
  • alert
  • switch
  • hidden_field
  • photo
  • gps
  • html_panel
