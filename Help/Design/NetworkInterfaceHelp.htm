When a user taps this field it tells your device to send and/or receive data between a set of device fields and a set of web data source fields. You define the details of the data interface in the Manage Data Page for native apps and in Manage Page Data for web apps.

Supported web data sources include:

Google Spreadsheet  - you can map the device fields to the Google Spreadsheet columns.

Web Services - you can map the device fields to the parameters of your web service methods.

SQL Database - you can map the device fields to your database tables and fields through a web service. ViziApps provides a free source code project you can install in your server or cloud instance.

If you don't know anything about web services or databases, you can ask our support team to help you by clicking on the Support button.
