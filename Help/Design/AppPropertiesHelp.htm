The Application Properties set properties for the whole app that include:

For All App Types:

App Description    This describes the app for your own needs.

Set Main Device Type for App    The choices include iPhone, iPad, Android Phone and Android Tablet. Custom canvas images are displayed for each selected device type that provide accurate frame displays with placement accuracy to 1 pixel on the final device page.

App Background  Choose among 67 attractive prepared backgrounds or upload your own custom image background for you smart phone. For iPad and Android tablets you can choose a custom background color but not a background image.

For Native App Types Only:

Do Action On App Open This will enable your app to get data from a web data source as soon as your app opens before the first page is shown.

See Images from Running App This will show all the images captured on device cameras taken by your app as it runs.

For Hybrid and Web App Types Only:

Add Custom Header HTML You can put any type of header HTML in your hybrid and web app including scripts, styling, Google Analytics, external web service calls, library plugins and others. Put all of your  HTML elements together in one custom header. You can assume a JQuery context for your scripts. You can also refer to your ViziApps fields and buttons in your javascript.

Set Page Transition Type Choose from fade, pop, flip turn, flow, slidefade, slide, slideup, slidedown, none. The default is fade.

For Web App Types Only:

URL Account Identifier This name is required for Web Apps to host your app uniquely in our hosting environment. For security purposes, we don't recommend using your account username.
