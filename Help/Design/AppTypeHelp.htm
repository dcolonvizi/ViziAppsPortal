A Native App runs on a mobile device using the native code of the device and has access to all the device resources such as the camera, GPS, accelerator, contact list, calendar and any embedded technology. A Native App is downloaded from an app store.

A Web App runs on the browser of a mobile device and does not have access to any of the device resources . A Web App is downloaded from a web host through a URL on a browser.

A Hybrid App shows the look and feel of the app on the browser of a mobile device but also has access to all of the device resources. A Hybrid App is downloaded from an app store.
