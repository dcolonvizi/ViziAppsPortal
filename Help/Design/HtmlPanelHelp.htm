The Html Panel enables you to enter and format any HTML type content such as text, images and links. You can enter the content directly or copy and paste the content from any other source.

After your content is entered, you can format it in many ways using the tools in the tool bar.

You can also preview the content and go into HTML mode to customized your content in pure HTML. This is only advised for experts in HTML code. In HTML mode , you can also add javascript code and CSS style code.

If you are trying to add image and links be certain that all hyperlink begin with "http" and have the full URL of the image or link you want. Shortcut URLs will not work. Shortcut URLs are often found in a company's website. For example if the website is "http://mydomain.com" and there is an image in the top folder named "myimage.png" , many websites will refer to it as "./myimage.png" which assumes the base URL of the website. This shortcut will not work here. However you can get the image to be displayed if you use the entire URL. In this example it would be "http://mydomain.com/myimage.png".
